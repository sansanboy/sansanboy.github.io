tabBar

客户端窗口底部的tab页面切换，只能配置最好两个、最多5个tab

属性说明：

属性 类型 必填 默认值 描述
color HexColor 是 tab上的文字默认颜色
selectedColor HexColor 是 tab上的文字选中时的颜色
backgroundColor HexColor 是 tab的背景颜色
boderStyle String 否 black tab上边框的颜色，仅支持black/white
list Array 是 tab的列表，详见list属性说明，最少2个、最多5个tab

list接受一个数组，数组中的每个项都是一个对象，属性值如下

属性 类型 必填 说明
pagePath String 是 页面路径，必须在pages中先定义
text String 是 tab上按钮文字
iconPath String 是 图片路径，icon大小限制为40kb
selectedIconPath String 是 选中时的图片路径，icon大小限制为40kb

webpack学习三

發表於 2016-10-05

我们不想运行webpack命令时都在后面输入一大堆参数，尤其是很大工程的时候，那么可以把所有的参数都放在配置文件中，配置文件的一个好处是可以方便以后的自动化部署

配置文件

  1. 在工程下面新建一个文件webpack.config.js `module.exports = {
    entry:"./entry.js", 
output:{
path:_dirname,
filename:"bundle.js"
},module{
   loaders:[
   {test:/\.css$/, loader:"style!css"}]}
    };
    `
  2. 运行 webpack
  3. 浏览器打开indexl.html ok

其它命令参数

  1. 显示进度条和颜色 webpack --progress --colers
  2. 文件变动时自己编译webpack --progress -- colers --watch

    开发服务器

  3. 安装 npm install webpack-dev-server -g
  4. 运行webpack -dev-server --progress --colors
  5. 运行 http://localhost:8080/webpack-dev-server/bundle

webpack学习一

發表於 2016-10-04

webpack顾名思义就是web开发打包工具 ，可以将img、css、js统一到一个文件中。

学习vuejs，学习组件时不想把所有的组件放到同 一个文件中，因此看demo等，是各个组件分别开发，最后打包到一个文件中。

按照官方教程
在本地学习。

入门

  1. 首先需要安装nodejs
  2. 安装： npm install webpack -g
  3. 新建一个空文件夹
  4. 新建文件entry.js document.write("It works")
  5. 新建文件index.html
<script type="text/javascript" src="bundle.js" charset="utf-8"></script>
  1. 运行命令 webpack ./entry.js bundle.js
  2. 浏览器打开index.html 将看 It works

模块导出

  1. 新建文件content.js module.exports="It works form content.js"
  2. 更新entry.js document.write(require("./content.js"))
  3. 重新运行 webpack ./entry.js bundle.js
  4. 浏览器打开，运行ok

webpack学习二

發表於 2016-10-04

webpack 打包css

安装

npm install css-loader style-loader -g

##开始

  1. 工程还为上一个工程，在工程下添加一个文件style.css
    `body{
    backgroud:yellow;
    }`
  2. 更新entry.js,添加一行 require("!style!css!./style.css") document.write(require("./content.js"));
  3. 运行webpack ./entry.js bundle.js
  4. 浏览器打开 index.html 查看，背景已经改变

    命令加入文件扩展

    css项我们肯定不想每次都那么些，所以可以在运行命令时绑定文件扩展
  5. 修改入口文件entry.js require("./style.css"); document.write(require("./content.js"))
  6. 运行命令webpack ./entry.js bundle.js --module--bind 'css=style!css' css文件绑定模块style
  7. 浏览器运行，ok

微信小程序配置

發表於 2016-09-24

pages

pages 设置页面路径，每一项代表页面的路径+文件名，数组的第一项代表初始页面

{"pages":
    [
        "page/index/index"
    ]
}

window

window 设置状态栏、导航条、标题、窗口背景色

属性 类型 默认值 描述
navigationBarBackgroundColor HexColor #000000 导航栏背景颜色，如”#000000”
navigationBarTextStyle String white 导航栏标题颜色，仅支持 black/white
navigationBarTitleText String 导航栏标题文字内容
backgroundColor HexColor #ffffff 窗口的背景色
backgroundTextStyle String dark 下拉背景字体、loading 图的样式，仅支持 dark/light
enablePullDownRefresh Boolean false 是否开启下拉刷新
