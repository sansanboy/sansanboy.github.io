java 和 php 谁更简单

发表于 2019-06-10 | 分类于 语言

最近在重新学习java，在14年的时候自己还是用Java的，在一个传统公司给银行做项目，当时用的框架是spring和mybites。记忆很深的是一大堆配置和过滤器。后来因为新公司用的是php，不知那来的勇气自学php了，后来到现在一直用php。直到最近，因为项目中使用到java，学习了一下java，使用spring boot。顺便和php框架做一个比较，发现java现在上手比php容易。

使用maven很容易搭建起一个demo来进行开发，然后配合注解，可以完全专注于开发，不需要关心框架的事情，对于业务开发来说是非常友好。当前比较流行的是golang和rust，很多php转做golang，建议也不要排斥java，golang是比较好，但是java还是有很大的市场，要淘汰会比较难。
